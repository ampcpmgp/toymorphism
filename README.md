# Toymorphism

Toy + Skeuomorphism

<https://ampcpmgp.github.io/toymorphism/>

## Development

### Install packages

```shell
npm install
```

### Develop components

```shell
npm run storybook
```

### Develop website

```shell
npm run dev
```

## Release

```shell
npm version [major | minor | patch | prerelease]
git push origin <tag_name>
```
