# Toymorphism

Toy + Skeuomorphism

<https://ampcpmgp.github.io/toymorphism/>

## Development

### Install packages

```shell
npm install
```

### Develop components

```shell
npm run storybook
```

### Develop website

```shell
npm run dev
```

## Pull Request

If edit `src/` files, enter below & commit it.

```shell
npx changeset # Enter changelog
```
